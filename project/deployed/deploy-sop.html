Connect Domain to Vercel
13–20 min

Add domain in Vercel dashboard
Go to vercel.com → your project (acp-website) → Settings → Domains. Add each of these one at a time:
ascensioncap.io ir.ascensioncap.io docs.ascensioncap.io portal.ascensioncap.io ops.ascensioncap.io
Vercel will show you the DNS records needed. Keep this tab open.
8
Set DNS records in Namecheap

While you wait: Add a payment card to Vercel so the free tier doesn't expire in 2 days. Go to Vercel → Settings → Billing → Add card. You won't be charged — the Hobby plan is free.
