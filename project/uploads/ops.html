Connect Domain → Vercel DNS
In Vercel → Project Settings → Domains → Add ascensioncap.io. Then in Namecheap DNS, set these records:
# Namecheap DNS Records @ A 76.76.21.21 www CNAME cname.vercel-dns.com ir CNAME cname.vercel-dns.com docs CNAME cname.vercel-dns.com portal CNAME cname.vercel-dns.com ops CNAME cname.vercel-dns.com
Propagation: 5–30 minutes. Then add each subdomain in Vercel dashboard under Domains.
